Young Finance Toolkit

Single-file, no-build web app that turns core personal-finance math into actionable decisions. Designed for classroom testing in Ocean Lakes High School’s Math & Science Academy senior project on financial decision-making of young adults. Runs entirely in the browser—no backend, no data leaves the device.

Live Demo
https://cyruslee2023.github.io/Convenient-Finance-Toolkit-for-young-adults/

Features

Multi-equation calculators

Future Value (FV), Present Value (PV)

APR ↔ APY conversions

Loan payment (amortized), fixed-payment payoff months, total interest

Inflation forward/backward purchasing power

Dynamic, context-aware suggestions

Feasibility check (payment vs. monthly interest)

Current-payment payoff estimate

Target-based guidance (e.g., “pay ~$X to finish in ≤ 24/36 months”)

Tiered advice for long/very-long payoff windows

APY quality coaching; inflation-aware notes

“Next steps” and guardrails

Planner

Read-only auto-suggestions + editable notes

Save/Load locally (localStorage)

Export plan to .txt

Quick Tests

PASS/FAIL checks for FV↔PV, APR↔APY, zero-rate loan payment, simple NPV check, helper round-trip

Reference sidebar (examples)

Curated links to broad, low-cost stock, bond, and cash funds from major providers

Not recommendations; for research and classroom discussion
